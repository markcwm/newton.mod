Newton.Openb3dmax: Functions Modinfo Source  

Newton wrapper for OpenB3DMax

Implements the declarations in Newton.Dynamics module.

Functions Summary

CreateNewtonCubeMesh Creates a Newton cube, attach a mesh, set the mass, true to create a stencil shadow.
CreateNewtonGround Creates a flat Newton ground from cube, set the width, height and depth.
CreateNewtonSphereMesh Creates a Newton sphere, attach a mesh, set the mass, true to create a stencil shadow.
CreateNewtonWorld Creates a Newton world object, set autofail (optional) to False to manually handle Null return.
ResetNewtonWorld Resets all the internal states of the Newton world.
UpdateNewtonWorld Updates the Newton world, set update_rate (optional) to affect the engine timestep.
